# ~~Gráficas y datos en diseño web responsive SVG, HTML, CSS y JavaScript (primera parte)~~ Diseño y programación: Datos para la propuesta

Cada estudiante debería tener, al final de la clase, al menos: 

- una hoja en Google SpreadSheet con los datos por visualizar
  
- una página html con un relato con el que se desplegarían tales datos

- un README.md con antecedentes y referentes, donde, por ejemplo: puede contar como antecedente esta página https://www.reuters.com/graphics/WALTDISNEY-FILM/SNOW-WHITE/gkvljloxavb/, pero no cuenta este sitio https://www.reuters.com/graphics/ -- Es claro que la página apunta a algo determinado, mientras el sitio no lo hace.

Para mayor claridad respecto del último punto, se recomienda revisar este PDF, que es un fragmento del *creando valor a través del diseño de servicios*: https://github.com/user-attachments/files/20396213/antecedentes-referentes.pdf
